**Air Pollution Analysis in India**

**1. Objective:**

a. Analyze **PM2.5** trends over time

b. Identify seasonal pollution patterns

c. Examine long-term pollution behavior

d. Interpret results from an economic perspective


**2. Dataset:**

Source: Kaggle Air Quality Data **(2017–2022)**

Features used:

a. Timestamp

b. PM2.5 concentration

c. Year, Month, Day

 
**3. Tools Used:**

a. Python (Pandas, Matplotlib, Seaborn)

b. Power BI (Dashboard creation, Interactive filtering and KPI tracking


**4. Methodology**

a. Data Cleaning (Removed missing values, Converted timestamp to datetime format)

b. Data Analysis (Time-series trend analysis, Monthly and yearly aggregation,Distribution analysis)

c. Visualization (Line charts for trends, Bar charts for yearly comparison, Histogram for distribution)

d. Dashboard Development (Built interactive Power BI dashboard, Added slicers for dynamic filtering, Included KPI for average PM2.5)


**5. Key Findings:**

a. Pollution peaks during winter months

b. PM2.5 levels remain consistently high

c. Seasonal variation is significant


**6. Economic Insight:**

a. Air pollution is a negative externality

b. Impacts productivity and healthcare costs


**7. Dashboard:**

Power BI dashboard included in repository


**8. Conclusion:**

Highlights need for policy intervention to reduce pollution
